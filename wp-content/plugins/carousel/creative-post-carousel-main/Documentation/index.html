Creative Post Carousel Elementor Addon

Installation Process

  1. Install Creative Post Carousel Elementor Addon using WordPress plugin installer or by uploading its files into the WordPress plugins directory.
  2. Activate the Creative Post Carousel Elementor Addon through WordPress Plugins menu.
  3. Click the Plugins => Add New
  4. Click Upload Plugin and then Choose the creative-post-carousel.zip from your computer then click on the Install Now button.
  5. After activating, it doesn't work without Elementor Page Builder plugin. Firstly, you should activate Elementor Page Builder.
  6. To activate newly installed plugin navigate the Plugins => Installed Plugins.
  7. Find the Creative Post Carousel Elementor plugin from the list and press/click Activate.
  8. Congratulations! plugin has been installed and activated on your website.
