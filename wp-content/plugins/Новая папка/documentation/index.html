How To Install


Download kontact.zip and upload it From Plugins-> add new. So Upload and active this plugin.

Display from WordPress Widget


To Display from WordPress Widget go to Appearance->Widgets and add Contact Widget which widget area you want to display contact from

Display from Elementor Widget


To Display from Elementor Widget You first need to active elementor plugin. Then Edit any page or post with elementor and Display Contact form 7 With drag and drop
